# LHFS Investor Match Engine

Real-time non-QM eligibility matching for Land Home Financial Services loan officers.

**Live site:** https://gfreesie.github.io/

## What it does

Matches borrower scenarios against investor guidelines across Luxury Mortgage, Acra Lending, NewRez, First National, and Deephaven — covering Bank Statement, DSCR, and ITIN programs.

Input FICO, LTV, transaction type, occupancy, and loan type to instantly see which investors have an eligible product, their max LTV, and minimum FICO floor.
